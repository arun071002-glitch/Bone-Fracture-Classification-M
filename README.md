# Bone Fracture Classification using Machine Learning

## 📌 Description

This project uses Machine Learning to classify bone fractures from X-ray images.

## ⚙️ Technologies Used

* Python
* Machine Learning
* OpenCV
* TensorFlow / Keras

## 📊 Features

* Image preprocessing
* Feature extraction
* Fracture detection
* Classification output

## 📁 Dataset

X-ray images of fractured and normal bones

## 🚀 Applications

* Medical diagnosis
* Hospital systems
* AI-based healthcare
